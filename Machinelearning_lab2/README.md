## Các công cụ chính:
- `CountVectorizer`: Để chuyển đổi văn bản thành các đặc trưng số.
-  `BernoulliNB` và `MultinomialNB`: Áp dụng các phân phối Naive Bayes khác nhau.
-  `train_test_split`: Chia dữ liệu thành tập huấn luyện và kiểm tra.
